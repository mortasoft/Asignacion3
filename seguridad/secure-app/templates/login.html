Estándares de Seguridad Aplicada.

Seguridad Implementada

Vulnerabilidad Mitigada

Este portal utiliza Sentencias Preparadas (Parameterized Queries) para neutralizar cualquier intento de SQL Injection.

Criptografía de Contraseñas

A diferencia del portal inseguro, aquí las contraseñas nunca viajan ni se almacenan en texto plano. Se utiliza BCrypt con Salt adaptativo.

Prueba de Penetración

Intenta usar el payload anterior:
' OR '1'='1
Verás que el sistema lo trata como una cadena literal, fallando la autenticación de forma segura.

Auditoría

El sistema rechaza peticiones incompletas y maneja errores sin revelar información sensible de la infraestructura de BD.
